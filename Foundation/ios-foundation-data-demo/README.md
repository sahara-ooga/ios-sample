# Data

## 概要
`Data`はバイトデータのラッパークラスです。

## 主要プロパティ

| プロパティ名 | 説明 | サンプル |
|-----------|------------|------------|
| `count` | バイト数を表す`Int` | `data.count` |
| `isEmpty` | 空かどうか | `data.isEmpty` |

## 主要メソッド

| メソッド名 | 説明 | サンプル |
|-----------|------------|------------|
| `base64EncodedData()` | `base64`エンコードする | `data.base64EncodedData()` |

## フレームワーク
Foundation.framework
